== Description ==

wpToWikipedia crea un widget che inserisce un link a wikipedia relativo al titolo dell'articolo, cerca ossia il titolo su wikipedia.


== Installation ==

Estrai il file zip ed inserisci il contenuto nella cartella wp-content/plugins/ del tuo wordpress e poi attivalo dalla pagina dei plugin. Amministralo poi dalla sezione widget.
